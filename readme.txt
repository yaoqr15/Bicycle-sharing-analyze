XXX_day.csv数据集以天为单位，XXX_hour.csv数据集以小时为单位，各特征含义如下：
    --ind：样本编号
    --dteday：日期
    --season：季节（1：春天；2夏天；3：秋天；4：冬天）
    --yr：年度（0:2011；1:2012）
    --mnth：月份（1至12）
    --hr：小时（0至23，在XXX_day.csv中不可用）
    --holiday:是否为假日（取自http://dchr.dc.gov/page/holiday-schedule）
    --weekday：星期几
    --workingday：如果不是假日或周末为1，否则为0.
    +weathersit：
      --1：晴，无云，局部多云
      --2：薄雾+多云，薄雾+碎云，薄雾+无云，薄雾
      --3：小雪，小雨+打雷+碎云，小雨+碎云
      --4：大雨，冰雹，雷雨+薄雾，雪+雾
    --temp：规范化摄氏温度（温度值除以最高温度41）
    --atemp：规范化体感摄氏温度（温度值除以最高温度50）
    --hum：规范化湿度（湿度值除以最高湿度100）
    --windspeed：规范化风速（风速值除以最高风速67）
    --casual：临时用户租赁单车数量
    --registered：注册用户租赁单车数量
    --cnt：租赁单车总数
